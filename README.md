# Análisis de Encuesta a Profesionales de Datos

## Objetivo del Proyecto
Desarrollo de un dashboard interactivo en Power BI para analizar las tendencias salariales, la demografía y la satisfacción laboral de más de 600 profesionales del sector de los datos.

## Herramientas Utilizadas
* **Power BI:** Visualización de datos y creación de reportes interactivos.
* **Power Query:** Limpieza de datos, estandarización de columnas (ej. rangos salariales) y cambio de tipos de datos.

## Conclusiones Clave
1. Los Científicos de Datos registran los salarios promedio más altos entre los encuestados.
2. Python destaca como el lenguaje de programación preferido por la mayoría de los profesionales.
3. Se observan variaciones salariales significativas dependiendo del país de residencia.
